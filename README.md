## Forker et personnaliser

1. **Fork** ce repository
2. Renommez-le en `votre-username.github.io`
3. Activez **GitHub Pages** dans Settings > Pages > Source: `main` / `root`
4. Modifiez `index.html` :
   - Section hero : nom, description, statistiques
   - Section about : votre parcours et timeline
   - Section projects : vos projets avec liens GitHub
   - Section skills : vos compétences et niveaux
   - Section contact : vos coordonnées et liens
5. Committez et poussez — votre site sera en ligne sous quelques minutes
